# 📱 Shop Điện Thoại

Website bán điện thoại được xây dựng bằng PHP & MySQL phục vụ mục đích học tập và nghiên cứu.

---

## 🚀 Công nghệ sử dụng

- PHP
- MySQL
- HTML
- CSS
- JavaScript

---

## ✨ Chức năng chính

- Đăng nhập / Đăng ký tài khoản
- Hiển thị danh sách sản phẩm
- Phân loại theo thương hiệu
- Tìm kiếm sản phẩm
- Trang quản trị Admin
- Thêm / Sửa / Xóa sản phẩm
